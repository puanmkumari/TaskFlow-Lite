# 🚀 TaskFlow Lite

A responsive **Task Management Web Application** built using **HTML, CSS, and Vanilla JavaScript (ES6)**. The app allows users to manage daily tasks with CRUD operations and stores data using **localStorage**.

## ✨ Features

* Add, Edit, Delete Tasks
* Mark Tasks as Complete
* Filter (All / Active / Completed)
* Data Persistence with localStorage
* Responsive Design
* Dark / Light Mode
* Form Validation

## 🛠️ Technologies

* HTML5
* CSS3
* JavaScript (ES6)
* localStorage
* DOM Manipulation

## 📂 Project Structure

taskflow-lite/

## 🚀 How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
